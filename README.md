# Python学习计划提醒系统 - QT可视化版本

## 概述

这是一个基于PyQt5的Python学习计划提醒系统可视化界面，将原有的命令行工具升级为功能完整的桌面应用程序。

## 功能特点

### 🎯 核心功能
- **3天Python速成计划**：包含详细的学习时间安排
- **可视化界面**：直观显示学习进度和计划
- **自动提醒**：在指定时间发送学习提醒
- **系统托盘**：支持最小化到系统托盘
- **跨平台通知**：使用QT系统通知，支持Windows/Linux/macOS

### 🖥️ 界面组件
1. **状态显示区**：显示当前学习进度和状态
2. **计划表格**：以表格形式展示今日学习计划
3. **控制按钮**：启动计划、开始/停止提醒、刷新、退出
4. **系统日志**：记录所有操作和提醒事件
5. **进度条**：直观显示学习进度（0-100%）

### ⚙️ 系统特性
- **配置保存**：自动保存学习计划开始日期到JSON文件
- **线程安全**：使用QT线程处理定时提醒，避免界面卡顿
- **错误处理**：完善的异常处理和用户反馈
- **托盘菜单**：右键系统托盘图标可快速操作

## 安装要求

### 必需依赖
```bash
pip install PyQt5 schedule
```

### 可选依赖（已集成，无需额外安装）
- 系统通知：使用QT内置通知系统
- 配置文件：使用Python标准json模块

## 使用方法

### 启动程序
```bash
# 方法1：直接运行QT版本
python study_reminder_qt.py

# 方法2：使用启动脚本（推荐）
python run_qt_reminder.py
```

### 操作步骤
1. **启动程序**：运行上述命令打开应用程序
2. **启动计划**：点击"🚀 启动计划"按钮
3. **查看计划**：程序会自动显示今天的学习计划
4. **开始提醒**：点击"⏰ 开始提醒"按钮启动自动提醒
5. **系统托盘**：关闭窗口可最小化到系统托盘

### 功能说明
- **启动计划**：设置学习计划的开始日期
- **开始提醒**：启动定时提醒系统
- **停止提醒**：停止所有提醒
- **刷新**：更新界面显示
- **退出**：关闭应用程序

## 技术架构

### 类结构
1. **StudyReminderSystem** - 核心逻辑类
   - 管理学习计划状态
   - 处理配置保存/加载
   - 计算学习进度

2. **MainWindow** - 主窗口类
   - 管理所有UI组件
   - 处理用户交互
   - 集成系统托盘

3. **ReminderThread** - 提醒线程类
   - 在后台运行定时提醒
   - 使用schedule库处理定时任务

4. **PlanStatus** - 状态枚举类
   - 定义学习计划的各种状态

### 数据流
```
用户操作 → MainWindow → StudyReminderSystem → 配置/状态更新
定时提醒 → ReminderThread → 通知系统 → 用户接收
```

## 故障排除

### 常见问题
1. **无法启动程序**
   - 检查Python版本（建议3.6+）
   - 运行 `python test_imports.py` 检查依赖
   - 确保已安装PyQt5和schedule

2. **界面显示异常**
   - 检查系统字体设置
   - 尝试调整窗口大小
   - 重启应用程序

3. **提醒不工作**
   - 检查系统时间设置
   - 确保程序在运行状态
   - 查看系统日志了解详情

4. **系统托盘不显示**
   - 某些Linux桌面环境需要额外配置
   - Windows/macOS通常支持良好

### 调试方法
```bash
# 查看详细错误信息
python study_reminder_qt.py 2>&1 | tee error.log

# 检查配置文件
cat study_reminder_config.json
```
